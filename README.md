# Pizza Order System(Code Lab's FDC Course)

## After installing or cloning package... 

### First Run

```
composer install
```

###  Copy the files from .env.example and create .env and then paste the copied files

```
cp .env.example .env or copy .env.example .env
```
### Generate application key
```
php artisan key:generate
```

### Migrate database
```
php artisan migrate
```

### Generate example datas
```
php artisan db:seed
```
### Link public folder and storage folder
```
php artisan storage:link
```

### Start Server
```
php artisan serve
```
